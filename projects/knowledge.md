## 📖 最佳实践库

### 代码规范

```markdown
# Java 代码规范

## 命名规范
- 类名：大驼峰 (UserService)
- 方法：小驼峰 (getUserById)
- 常量：全大写 (MAX_RETRY_COUNT)
- 变量：小写 + 下划线 (user_count)

## 注释规范
- 类注释：说明类的用途
- 方法注释：说明参数、返回值、异常
- 行注释：解释复杂逻辑

## 异常处理
- 捕获具体异常
- 记录异常日志
- 不吞异常
- 不抛受检异常
```
